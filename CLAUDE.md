# Claude Code guidance for todomate-cli

## This is a public repo

Every commit message, comment, and file is visible on GitHub. Keep these out of all public-facing content:

- Internal infrastructure names (database technologies, cloud service names, internal service names)
- Internal API details beyond what's in the public OpenAPI spec
- Any secrets, keys, or internal URLs

Write commit messages and comments as if a stranger on the internet will read them — because they will.
